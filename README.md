# Ecommerce Template on ReactJS

-   Built on ReactJS, NodeJs, ExpressJs & MongoDb.


## How to run

`cd algorithm-visualiser`
Firstly, start backend:
`cd backend`
`npm install`
`node server.js`

To start frontend : 
`cd frontend`
`npm install`
`npm start`

Now visit : http://localhost:3000/
